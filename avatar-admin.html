Security & Tips

Upload security
Every image goes through the /api/avatar/upload-skin endpoint. It verifies: admin token, MIME whitelist (PNG/JPEG/GIF/WebP — no SVG), magic bytes match declared type, max 512 KB, clean base64 re-encode before DB storage. The manage-item endpoint will reject raw imageData blobs.

Skin → Custom Image
Your image wraps all 6 faces of the body cube as a texture. Square images work best. PNG with a transparent background looks cleanest.

Custom Image Overlay
A flat transparent plane placed on the avatar. Use sliders to position it — great for logos, face overlays, badges.

File size
Images stored as base64 in D1. Compress first at squoosh.app — aim under 150 KB.
